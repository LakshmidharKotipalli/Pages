03A. Learning Journey (4 Levels)

Level 1: Don't Break It (0-6 months)

Goal: Send an image via an API and get a response without crashing the system.

  • Base64 encoding images safely.
  • Understanding payload limits (you can't send a 40MB raw 8K image).
  • Prompting VLMs: "Look at this image and tell me X".

Level 2: Comfortable Practitioner (6mo-2yr)

Goal: Running local VLMs (vLLM, Ollama), handling chunking and basics of RAG.

  • Using multimodal-vLLM to host Qwen-VL or LLaVA.
  • Understanding how number of image patches eats context window tokens (e.g., a 1024x1024 image = ~1024 tokens).
  • Basic structured output (JSON generation) from receipts.

Level 3: Problem Solver (2-5yr)

Goal: Parameter Efficient Fine Tuning (LoRA) and layout parsing.

  • Freezing the vision encoder and LLM, only training the projector layer.
  • Solving the "small text" problem via tiling/AnyRes processing.
  • Managing GPU KV-cache explosion from images.

Level 4: Architect's Mind (5-15yr)

Goal: Ground-up pre-training strategies, latent space alignments.

  • Designing continuous vs discrete tokenization structures (VQ-VAE vs SigLIP grids).
  • Architecting cross-attention layers that inject vision directly into deeper transformer layers (like Flamingo) rather than just prefixing visual tokens.
  • Handling catastrophic forgetting during multimodal alignment.
