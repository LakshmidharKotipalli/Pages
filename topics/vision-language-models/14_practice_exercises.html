14. Practice Exercises

Warm-up

  1. Base64 encode a 512x512 image of a cat and prompt a local LLaVA instance: "Describe the fur color." (Tests basic payload assembly).
  2. Calculate the token consumption if you pass a 1024x1024 image to a ViT-L/14 model (Assuming 14x14 pixel patches, no compression).

Core Engineering

  1. Take a receipt image. Use an open-weights VLM (e.g., Qwen-VL) to extract the Total Amount into strictly formatted JSON. Then resize the image to 224x224 and run it again. Observe the failure.
  2. Implement a basic System Prompt KV Cache preservation script using vLLM so subsequent questions about the same image take < 200ms TTFT.

Advanced Architecture

  1. Using PyTorch, instantiate a HuggingFace CLIP model and write the cosine-similarity matrix between 5 animal images and 5 caption strings to see the modality alignment in action.
  2. Write a dynamic AnyRes image cropper that splits a 1920x1080 UI screenshot into overlapping 336x336 tiles to feed into a standard VLM projector. Combine the final tokens into a 2D grid payload for the LLM.
