Getting Started in 60 Seconds

Install and start using Claude Code immediately on any platform.

Installation

Choose Your Platform

🍎

macOS / Linux

curl -fsSL https://claude.ai/install.sh | bash

Or via Homebrew:

brew install --cask claude-code # Upgrade later with: brew upgrade claude-code
🪟

Windows

irm https://claude.ai/install.ps1 | iex

Or via WinGet:

winget install Anthropic.ClaudeCode # Upgrade later with: winget upgrade Anthropic.ClaudeCode
💻

VS Code / Cursor

Open Extensions (Cmd+Shift+X / Ctrl+Shift+X)

Search for anthropic.claude-code

Install and activate via Cmd+Shift+P / Ctrl+Shift+P

🖥️

Desktop App

Download from macOS (Intel & Apple Silicon) or Windows (x64). Requires a paid Claude subscription.

🌐

Web

Go to claude.ai/code — run Claude Code on Anthropic's cloud infrastructure in isolated VMs. No local installation needed.

🔧

JetBrains

Install the Claude Code plugin from the JetBrains Marketplace. Works with IntelliJ, PyCharm, WebStorm, and all JetBrains IDEs.

First Session

Your First 5 Minutes

Navigate & Launch

cd your-project claude

Initialize Project Memory

Create a CLAUDE.md that gives Claude persistent memory of your project:

> /init

Explore Your Codebase

> give me an overview of this codebase > explain the main architecture patterns used here > how is authentication handled?

Make Your First Change

> add input validation to the user registration endpoint. write tests and run them to verify.
💡 Pro Tip

Run /init in every new project. It's the single most impactful setup step. Claude will analyze your project structure and generate a tailored CLAUDE.md with conventions, commands, and patterns.

🔑 Authentication

You'll need a Claude subscription (Pro, Team, or Enterprise) or an Anthropic Console API key. Third-party providers like AWS Bedrock and Google Vertex AI are also supported.
