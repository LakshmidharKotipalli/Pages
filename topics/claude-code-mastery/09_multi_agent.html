Multi-Agent Development

Run multiple Claude sessions simultaneously. Worktrees, agent teams, parallel sessions, and cross-device workflows.

Parallel Sessions

Git Worktrees — Isolated Parallel Work

Each Claude session gets its own isolated git worktree — no merge conflicts between parallel agents working on the same project.

# Start Claude in a named worktree # Creates .claude/worktrees/feature-auth/ with a new branch claude --worktree feature-auth # Start another session in a separate worktree claude --worktree bugfix-123 # Auto-generate a worktree name claude --worktree # Creates something like "bright-running-fox"

How It Works

  • Each worktree lives in <repo>/.claude/worktrees/<name>
  • A new branch worktree-<name> is created automatically
  • Full filesystem isolation — no conflicts between sessions
  • Cleanup available for completed worktrees

Use Cases

  • Work on a feature while a bug fix runs in parallel
  • Test Opus vs Sonnet on the same problem simultaneously
  • Run code review in one session while implementing in another
  • Explore alternative implementation strategies concurrently
Experimental

Agent Teams

Coordinate multiple Claude instances with shared tasks, messaging, and a team lead. One agent writes code, another reviews, a third writes tests.

🏗️ Code Review Pattern (Manual)

Session A: "Implement a rate limiter for our API endpoints"
Session B: "Review the rate limiter implementation in @src/middleware/rateLimiter.ts. Look for edge cases, race conditions, and consistency with our existing middleware patterns."
Session A: "Here's the review feedback: [Session B output]. Address these issues."

🖥️

Desktop App Sessions

Manage multiple local sessions visually. Each session gets its own isolated worktree. See diffs and changes in a graphical interface.

🌐

Claude Code on the Web

Run on Anthropic's secure cloud infrastructure in isolated VMs. No local resources consumed. Start tasks from anywhere.

🤖

Agent Teams (Experimental)

Automated coordination of multiple sessions with shared tasks, inter-agent messaging, and a team lead that manages the overall plan.

Cross-Device

Work From Anywhere

/teleport — Web/iOS → Terminal

Start a long-running task on the web (claude.ai/code) or iOS app, then pull it into your terminal with /teleport when you're back at your desk.

/desktop — Terminal → Desktop App

Hand off a terminal session to the Desktop app with /desktop for visual diff review, graphical file comparisons, and a richer editing experience.

@Claude in Slack

Mention @Claude in Slack with a bug report or feature request. Claude can investigate, implement a fix, and create a pull request — all triggered from team chat.

Chrome Extension

Use the Claude Code Chrome extension for browser-based interactions, visual comparison, and web-page-context tasks.
