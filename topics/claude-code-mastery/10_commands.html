Commands Reference

Complete reference for slash commands, keyboard shortcuts, and CLI flags.

Slash Commands

In-Session Commands

Command Description
/init Generate a CLAUDE.md for your project with conventions, commands, and patterns
/clear Reset context completely — use between unrelated tasks
/compact [instructions] Compress context with optional focus guidance (e.g., /compact Focus on API changes)
/rewind Open checkpoint menu to restore previous conversation and code state
/permissions Configure tool allowlists and domain access for URLs
/sandbox Enable OS-level isolation — restricts filesystem and network access
/hooks Configure automated event triggers (PreToolUse, PostToolUse, etc.)
/agents View and manage available subagents
/plugin Install community plugins
/rename [name] Name your session for easy resuming (e.g., /rename "oauth-migration")
/teleport Move a session from web/iOS to your terminal
/desktop Hand off current session to the Desktop app for visual review
/insights Usage reports — patterns, friction points, and suggested improvements
/commit-push-pr Commit changes, push, and create a pull request in one step
Keyboard Shortcuts

Keyboard Controls

Shortcut Action
Esc Stop Claude mid-action (context preserved, redirect immediately)
Esc + Esc Open rewind menu — restore checkpoints or summarize from a point
Shift+Tab Toggle Auto-accept mode — let Claude work autonomously
Ctrl+G / Shift+Tab×2 Toggle Plan Mode — read-only analysis, no code changes
Cmd+Shift+X Open VS Code Extensions panel (for installation)
Cmd+Shift+P Open VS Code Command Palette (to activate Claude Code)
CLI Flags

Command-Line Options

Flag Description
claude -p "prompt" Headless mode — process prompt and exit
--output-format json JSON output for scripting (also: text, stream-json)
--continue Resume the most recent conversation
--resume Select from recent conversations to resume
--worktree [name] Start in an isolated git worktree (auto-name if no name given)
--from-pr <number> Start a session pre-loaded with a PR's context
--allowedTools "..." Restrict tools Claude can use (e.g., "Edit,Bash(git commit *)")
--dangerously-skip-permissions Skip all permission prompts (sandbox only!)
--verbose Show detailed output for debugging
claude mcp add Add an MCP server connection
Quick Prompts

Common One-Liners

# Explore a codebase claude "give me an overview of this codebase" claude "trace the login process from front-end to database" # Bug fixing claude "I'm seeing an error when I run npm test" claude "suggest a few ways to fix the @ts-ignore in user.ts" # Tests claude "find functions in NotificationsService.swift not covered by tests" claude "write tests for the auth module, run them, fix failures" # Refactoring claude "refactor utils.js to use ES2024 features, maintain same behavior" claude "find deprecated API usage in our codebase" # Documentation claude "add JSDoc comments to undocumented functions in auth.js" # Git claude "commit with a descriptive message and open a PR" claude "summarize the changes I've made to the authentication module"
