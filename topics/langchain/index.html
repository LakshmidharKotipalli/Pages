🔗 LangChain Ecosystem

LangChain is more than just a framework—it's a complete ecosystem for building, deploying, and observing LLM applications.

🦜 LangChain (Core)

The Framework

Python and JavaScript libraries for building LLM applications. The orchestration layer that transforms raw LLM capabilities into production-ready apps.

  • Chains, Agents, and Prompts
  • Memory and Tools
  • Document loaders and splitters
  • Vector store integrations

🛠 LangSmith

Developer Platform

Debug, test, and monitor your LLM applications. Trace every step of your chains, evaluate performance, and iterate faster.

  • Full trace visibility
  • Evaluation and testing
  • Annotation and feedback
  • Performance metrics

🚀 LangServe

Deployment Platform

Deploy any LangChain chain as a REST API in one line. Built on FastAPI with automatic type validation and async support.

  • One-line chain deployment
  • Automatic API generation
  • Async and streaming support
  • Production-ready monitoring

Before starting this learning journey, you should have:

  • Basic Python knowledge - Functions, classes, decorators, async/await
  • Fundamental ML concepts - What a model is, embeddings, basic vector operations
